# Public Discovery

Public discovery describes how StegTalk exposes local entity discovery records, indexes those records, searches them, and demonstrates the discovery path.

Runtime fields:

- public discovery record
- discovery index
- search query
- matched terms
- scored result envelope
- demo trace

StegTalk owns discovery record shape, discovery index behavior, search result shape, and local discovery demo trace.

Overlap:

- Site may publish public-facing explanations or demos.
- Admissibility may evaluate whether a discovery record can be used for a transition.
- StegGuardian may evaluate whether discovery exposes account, guardian, or device-sensitive boundaries.

Boundary: public discovery readiness does not imply public deployment authority, production directory service, or external identity federation.
